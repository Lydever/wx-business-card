# wx-business-card

基于微信原生小程序开发的一个移动端简易名片编辑小程序，实现基本信息输入，并保存到手机相册

# 核心功能
- 根据用户输入的信息，绑定显示到模板上
- 实现模板图片保存
- 实现取消操作
